Champagne Party

This project is a space for tasting champagne.
Draped in a style reminiscent of Parisian waiters,
it features champagne in 3 modules and a quiz.

More details

Using a set of pre-drilled cleats of pre-defined sizes,
our proposal consists of 3 modules:
the first holds bottles on the right,
the second holds glasses at the bottom and finally,
on the left, a module for exchanging and holding the ice cube tray.
